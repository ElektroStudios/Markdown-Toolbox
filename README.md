## 👋 Introduction

This project is a simple, lightweight Windows desktop application built with WinForms and VB.NET. It centralizes a selection of essential Markdown web utilities into a single, distraction-free desktop interface, removing the need to manage multiple browser tabs for routine editing and conversion tasks.

The application integrates direct, dedicated access to the following resources:

 - [StackEdit.io](https://stackedit.io/app#): The best online markdown editor out there.
 - [Markdown to HTML Converter](https://markdowntohtml.com/#converter): Quickly and easily converts Markdown to HTML.
 - [HTML to Markdown Converter](https://www.browserling.com/tools/html-to-markdown): Efficiently converts HTML to Markdown.
 - [Markdown Table Converter](https://markdown-convert.com/en/tool/table): Facilitates the conversion of tables from various formats to Markdown.
 - [Markdown Guide](https://www.markdownguide.org/basic-syntax/): A comprehensive guide to Markdown syntax.
 - [ChatGPT](https://chat.openai.com/): Because why not?, ChatGPT can help you improve your Markdown document, give you ideas, etc.

## 💡 Motivation

When writing documentation or managing repository files, switching between different online converters, editors, and syntax cheatsheets quickly litters your browser with open tabs. It is easy to lose focus or accidentally close the tool you were working with. I built this humble container to keep all essential Markdown resources neatly organized under one roof, providing an isolated desktop environment that helps keep your workspace clean and efficient.

##### ⚡ The Real Question
###### Why clutter your web browser with endless tabs for writing, converting, and formatting text when you can access every Markdown utility you need from a single desktop dashboard?.

## 🚀 Getting Started

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/Markdown-Toolbox/releases/latest)**.
2. Download the latest `.zip` archive or the `.exe` setup installer, depending on your preference.
3. If you downloaded the `.zip` archive, extract its contents to your preferred directory.
   <br> 
   If you downloaded the `.exe` file, run it and follow the installation wizard.
4. Run the executable file to launch the application.
